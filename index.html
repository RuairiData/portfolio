Project 4 — Energy Forecast ML Dashboard

Python • Plotly Dash • time-series ML • model comparison

Why it’s relevant for analytics / BI roles

This dashboard turns a forecasting notebook into a decision tool: it lets stakeholders compare classical baselines (SARIMA/SARIMAX) against neural sequence models (LSTM) under consistent, time-based evaluation. Users can toggle traces to reduce clutter, inspect holdout behaviour, and use RMSE/MAE summaries to rank models without losing the visual “why”.

What I’d highlight in interviews: time-aware train/test/holdout splits, horizon-based evaluation (24h / 72h / 168h), transparent metrics, and interaction that changes what is shown rather than changing the visual grammar.

Project 5 — New Dwelling Completions Dashboard (Ireland)

Python • Plotly Dash • geo-viz • interactive filtering

Why it’s relevant for BI / reporting roles

This dashboard is a compact “BI-style” product: a single filter (Quarter) drives consistent KPI cards, a county map view, and selectable chart views (bar / pie / line). It demonstrates the same skills employers look for in Power BI roles—clear KPIs, cross-filtering logic, and multiple stakeholder-friendly views of the same measure.

Skills demonstrated: metric definitions, interactive controls, ranking (top/lowest county), geo visualisation, and chart switching for different analytical questions.
