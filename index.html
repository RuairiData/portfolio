Project 3 — Tik Box (Habit Analytics)

KPI design • time intelligence • targets vs actual • insights

Why it’s relevant for Power BI roles

Tik Box is a KPI reporting + insights product: it tracks completion over time (YTD and cumulative), measures progress vs weekly targets, and highlights patterns (best/most skipped day, habit pairings). It mirrors the “overview → drilldown” journey you build in MI/BI dashboards.

Outcome: Clear progress-to-target reporting with consistent metric definitions and a drill-down path from overview insights to habit-level detail.
